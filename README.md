# AI SDK 6 Skill

A Claude Code skill for building agents with Vercel AI SDK 6.

## Install

```bash
npx skills add https://github.com/francostan/ai-sdk-6-skill -g -y
```

## What's inside

- ToolLoopAgent patterns
- Human-in-the-loop safety
- MCP integration
- RAG with reranking
- DevTools debugging
- Common error fixes

## Use it

```
/ai-sdk-6-skill
```

Ask things like "how does ToolLoopAgent work?" or "show me needsApproval pattern"
